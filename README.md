# 🚀 My Obsidian Knowledge Management Setup

这是我的个人 Obsidian 配置，专为 **计算机专业学习（后端开发/算法）** 与 **高效笔记输出** 打造。

## 🎨 主题与视觉体系

### **Minimal Theme**
本项目基于 [Minimal](https://github.com/kepano/obsidian-minimal) 主题构建。
* **设计理念**：极致简约，减少视觉干扰，让内容回归核心。
* **深度定制**：通过自研 CSS 片段修正了 Minimal 在处理大段代码块与多级标题时的间距局促问题。

---

## 🔌 核心插件介绍

| 插件名称 | 用途与配置亮点 |
| :--- | :--- |
| **Code Styler** | **代码块颜值担当**。深度定制 Java 代码块渲染，隐藏了实时预览下的反引号行。 |
| **Style Settings** | **图形化配置中心**。配合自定义 CSS 变量，实现无需改代码即可微调 UI。 |
| **Hover Editor** | **多任务神器**。将悬浮预览转化为全功能编辑器，极大方便算法复习。 |
| **Minimal Theme Settings** | 官方配套工具。用于快速切换 Minimal 的各种内置预设模式。 |
| **Image Toolkit** | **图片增强**。支持图片的点击放大、缩放与旋转，适合查看架构图。 |

---

## ✨ 核心改进（基于自定义 CSS 片段）

### 1. 标题节奏感优化 (`head-height.css`)
* **场景化间距**：自动识别标题上方元素。若标题紧跟在**代码块**后，会强制推开呼吸空间。
* **相邻标题对冲**：当 H1-H6 连续排列时，自动消除多余的 margin 叠加。

### 2. 段落间距体系 (`spacing.css`)
* **代码块优化**：解决了 Code Styler 插件导致的垂直留白过大问题，比例更加和谐。
* **列表衔接补丁**：优化了有序列表（1. 2. 3.）与下方段落的距离。
