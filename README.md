\# 🚀 My Obsidian Knowledge Management Setup



这是我的个人 Obsidian 配置库，专为 \*\*计算机专业学习（后端开发/算法）\*\* 与 \*\*高效笔记输出\*\* 打造。核心改进集中在视觉层级的深度定制与插件联动，解决了复杂笔记场景下的排版痛点。



\## 🎨 主题与视觉体系



\### \*\*Minimal Theme\*\*



本项目基于 \[Minimal](https://github.com/kepano/obsidian-minimal) 主题构建。



&#x20; \* \*\*设计理念\*\*：极致简约，减少视觉干扰，让内容回归核心。

&#x20; \* \*\*深度定制\*\*：通过自研 CSS 片段修正了 Minimal 在处理大段代码块与多级标题时的间距局促问题。



\-----



\## 🔌 核心插件介绍



| 插件名称 | 用途与配置亮点 |

| :--- | :--- |

| \*\*Code Styler\*\* | \*\*代码块颜值担当\*\*。我深度定制了 Java 代码块的渲染效果，隐藏了实时预览下的反引号（` ``` `）占位，并优化了行高与内边距。 |

| \*\*Style Settings\*\* | \*\*图形化配置中心\*\*。配合自定义 CSS 变量，实现了无需改代码即可微调 UI 颜色与组件尺寸的功能。 |

| \*\*Hover Editor\*\* | \*\*多任务神器\*\*。将悬浮预览转化为全功能编辑器，极大方便了在复习算法（如 Dijkstra）时快速查阅相关 Java 基础知识。 |

| \*\*Minimal Theme Settings\*\* | 官方配套工具。用于快速切换 Minimal 的各种内置预设模式（如 Focus Mode）。 |

| \*\*Image Toolkit\*\* | \*\*图片增强\*\*。支持图片的点击放大、缩放与旋转，适合查看复杂的架构图或流程图。 |

\## ✨ 核心改进（基于自定义 CSS 片段）



\### 1\\. 标题节奏感优化 (`head-height.css`)



&#x20; \* \*\*场景化间距\*\*：自动识别标题上方元素。若标题紧跟在\*\*代码块\*\*后，会强制推开呼吸空间，避免视觉拥挤。

&#x20; \* \*\*相邻标题对冲\*\*：当 H1-H6 连续排列时，自动消除多余的 margin 叠加，保持紧凑。



\### 2\\. 段落间距体系 (`spacing.css`)



&#x20; \* \*\*代码块瘦身\*\*：解决了 Code Styler 插件导致的垂直留白过大问题，让代码块与正文比例更加和谐。

&#x20; \* \*\*列表衔接补丁\*\*：优化了有序列表（1. 2. 3.）与下方段落的距离，确保算法步骤逻辑分明。



\### 3\\. 编辑模式“无痕化”



&#x20; \* 实现了在实时预览模式下自动隐藏 Markdown 标记符号，提供近乎于阅读模式的沉浸式撰写体验。

\## 🛠️ 如何复刻我的配置



1\.  克隆本仓库到本地。

2\.  将其中的 `.obsidian` 文件夹覆盖到你的库根目录。

3\.  在 `设置 -> 外观 -> CSS 代码片段` 中，确保 `spacing` 和 `head-height` 已启用。

4\.  \*\*注意\*\*：部分样式依赖 \[Code Styler](https://www.google.com/search?q=https://github.com/mayurans7/obsidian-code-styler) 插件，请确保已在社区插件中安装。
